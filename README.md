# 🏥 Public Health Intelligence Prompts Library

A comprehensive collection of AI prompts designed specifically for public health intelligence applications. This library provides ready-to-use prompts for epidemiologists, public health analysts, surveillance specialists, and other public health professionals.

## 📚 Library Structure

### Core Prompt Categories

| Category | Description | Key Applications |
|----------|-------------|------------------|
| **[Epidemiology](./prompts/epidemiology/)** | Disease investigation and analysis | Outbreak investigations, case definitions, contact tracing |
| **[Surveillance](./prompts/surveillance/)** | Health monitoring systems | Syndromic surveillance, data quality, early warning |
| **[Data Analysis](./prompts/data-analysis/)** | Statistical methods and interpretation | Descriptive statistics, regression, time series |
| **[Reporting](./prompts/reporting/)** | Communication and documentation | Situation reports, policy briefs, technical summaries |
| **[Risk Assessment](./prompts/risk-assessment/)** | Health threat evaluation | Hazard identification, vulnerability analysis |
| **[Policy](./prompts/policy/)** | Public health policy development | Policy analysis, evidence synthesis, implementation |
| **[Emergency Response](./prompts/emergency-response/)** | Crisis management | Emergency planning, incident command, coordination |
| **[Community Health](./prompts/community-health/)** | Population health improvement | Health promotion, community engagement, education |
| **[Training & Education](./prompts/training-education/)** | Workforce development | Curriculum design, case studies, competency assessment |

## 🚀 Getting Started

### Quick Start Guide

1. **Browse Categories**: Navigate to the relevant category folder for your use case
2. **Select Prompts**: Choose prompts that match your specific needs
3. **Customize**: Adapt prompts with your specific context and data
4. **Apply**: Use with your preferred AI assistant or language model
5. **Validate**: Always review AI outputs with domain expertise

### Example Usage

```markdown
# Using an Outbreak Investigation Prompt

1. Go to `/prompts/epidemiology/outbreak-investigation.md`
2. Copy the prompt template
3. Fill in your specific outbreak details:
   - Disease: Salmonellosis
   - Location: County X
   - Cases: 45 confirmed cases
4. Submit to your AI assistant
5. Review recommendations with your epidemiology team
```
